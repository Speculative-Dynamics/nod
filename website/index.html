Privacy is architecture.

Runs entirely
on your phone.

Nod has no backend. There's no server to leak, no account to delete, no API key to revoke. Every word you type stays in a local database on the device, read only by a model running on the same chip. iCloud backup for the conversation is off by default unless you turn it on.

We can't see what you write. Not because we promise — because Nod has no backend receiving your conversation.

  • Default engine Apple FoundationModels Apple Intelligence's on-device LLM. Runs the second you open the app — no download, no network.
  • Alternative engines Qwen 3, Qwen 3.5, Gemma 4 Open-weights models via MLX Swift. Switch in settings — Nod downloads the selected model once (2–3 GB, Wi-Fi by default), then runs on-device forever.
  • Memory On-device fact extraction Apple FoundationModels extracts structured memories from each turn. Browse them in the sidebar; delete any one of them, or all of them.
  • Storage Local SQLite GRDB.swift, encrypted at rest by iOS. Tap "Start fresh" and it's gone. iCloud backup is optional and off by default.
  • Analytics None No telemetry, no tracking pixels, no third-party SDKs.
